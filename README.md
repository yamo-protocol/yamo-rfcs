# YAMO RFCs

This repository contains standards-track proposals for the YAMO protocol.

## Purpose
RFCs define:
- Protocol semantics
- Block format changes
- Consensus rules
- IPFS bundling standards
- Multi-agent workflows
- Extensions and integrations

## How to Submit an RFC
1. Copy `template.md`
2. Name it `RFC-xxxx-title.md`
3. Open a PR
4. Participate in discussion
5. Editors will advance the RFC through the lifecycle

## Status Labels
- Draft
